версия java: 21.0.1

Сборка jar архива:
javac *.java
jar cfe util.jar Programm *.class


Примечания:
Файлы должны лежать возле jar архива.
Если указанного текстогого файла возле jar архива нет, то вылезет ошибка об отсутствии файла.

При использовании опции -o путь может быть как относительный, так и полный.
После опции -o должен идти путь, иначе вылезет ошибка, о некорректном пути.
Путь, в котором есть каталог с пробелом нужно написать в двойных кавычках.
При написании несуществующего пути вылезет ошибка о некорректном пути.

При опции -a прошлые значения из файлов в статистике не учитываются.

После опции -p должен идти префикс. Если Префикс отсутствует, то утилита возьмет следующий аргумент после -p.

При указании любой опции больше 1 раза вылезет ошибка о некорректном вводе опций утилиты.

В статистике сумма, минимальный, максимальный элементы, среднее арифметическое высчитываются из объединения int и float элементов.
Если указать сразу опции -f и -s, то выведется только полная статистика
Если элементов int и float в файлах не было, то сумма, среднее арифметическое, минимальный и максимальный элементы выводятся в статистике нулями
Если элементов string в файлах не было, то макс и мин длина строк выводятся в статистике нулями

Файлы указывать без .\




Запуск утилиты:
java -jar util.jar <опции> <файлы>
